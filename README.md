# Randomite

### Additions

## Blocks

* Randomite Ore
    + Variants for stone types
    + Drops either eggs, coal, raw iron, raw gold, lapis, redstone, diamonds or crude steel randomly
    + Generates at any Y level commonly

## Requirements

+ BTA 7.3_04
+ Babric
+ Halplibe
